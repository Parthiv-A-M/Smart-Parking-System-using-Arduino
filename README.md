# Smart Parking System Using Arduino

## Components Used
- Arduino UNO
- IR Sensors
- Servo Motor
- Breadboard
- Jumper Wires
- USB Cable
- Toy Cars

---

## Circuit Connections

| Component | Arduino Pin |
|-----------|-------------|
| Entry IR Sensor | D2 |
| Slot 1 IR Sensor | D3 |
| Slot 2 IR Sensor | D4 |
| Slot 3 IR Sensor | D5 |
| Slot 4 IR Sensor | D6 |
| Servo Motor Signal | D9 |
| IR Sensor VCC | 5V |
| IR Sensor GND | GND |
| Servo VCC | 5V |
| Servo GND | GND |

---

## Features
- Automatic gate control
- Parking slot detection
- Real-time vehicle monitoring
- Smart parking automation
- Parking full indication

---

## Software Used
- Arduino IDE
- Embedded C Programming
